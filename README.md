# blockchain-tracker
A blockchain market tracking app. Example implementation of reactive clean architecture and testing.

## Tech Stack
- Kotlin<br>
- MVVM<br>
- Clean Architecture<br>
- Dagger2<br> 
- Repository<br>
- RxKotlin<br>
- Data Binding<br>
- Arch Components<br>
- Espresso Test<br>
- Unit Test<br>
- Rotation Support
